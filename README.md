# PlantUML Generator from Textual Scenarios

## Описание

Проект предназначен для автоматического преобразования текстовых формализованных сценариев бизнес-процессов в корректный код PlantUML. Он позволяет:

- Генерировать UML диаграммы последовательностей (sequence diagrams) и диаграммы действий (activity diagrams) из текстовых сценариев.
- Обрабатывать вложенные условия (`alt / else`) и циклы (`loop`).
- Автоматически объявлять всех участников процесса, включая EmailNotifier.
- Поддерживать строгий корпоративный стиль кода UML без лишних пояснений или комментариев.

Проект разработан для использования в крупных организациях для документирования и автоматизации процессов.

---

## Возможности

- Генерация чистого PlantUML из текстового сценария.
- Поддержка альтернативных ветвей, циклов и вложенных действий.
- Совместимость с любыми инструментами, поддерживающими PlantUML (например, PlantUML Server, IntelliJ Plugin, VS Code).
- Полностью детерминированная генерация (без случайного мусора).
- Возможность тестирования и настройки системного промпта для разных сценариев.
